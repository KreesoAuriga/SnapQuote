# SnapQuote

This codebase is based on Jason Watmore's dotnet registration login api project with vulnerabilities added to replicate a real-world case of a data breach, for use in testing the ability of human and/or AI coding agents to identify the vulnerabilities. A slightly different version of this codebase for use in a docker container from pre-built binaries is at https://github.com/KreesoAuriga/SnapQuoteTestable and is designed for use in evaluating black-box pentesting ability.

Post an issue with contact information to acquire a list of the specific vulnerabilities added. They are not explicitly described here to make it difficult for human/AI testees to 'cheat'.
